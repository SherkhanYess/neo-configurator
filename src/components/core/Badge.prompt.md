Tiny status / category label. Not interactive — use Tag for selectable chips.

```jsx
<Badge tone="champagne">Premium</Badge>
<Badge tone="success" dot>В наличии</Badge>
<Badge tone="accent">VVS1</Badge>
```

Tones: `neutral`, `accent`, `success`, `warning`, `danger`, `champagne` (rare premium highlight). `dot` adds a status dot.
